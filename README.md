# 🎵 Tempo Bot

> ✨ **Помоги себе избежать выгорания всего за 5 минут в день**

Telegram-бот для баланса работы и отдыха с AI-анализом, аудио-практиками и готовыми материалами.

## 🚀 Быстрый старт

```bash
# 1. Установите зависимости
pip install -r requirements.txt

# 2. Создайте .env файл
cp .env.example .env

# 3. Заполните токены в .env
TELEGRAM_TOKEN=your_token_here
OPENAI_API_KEY=your_key_here

# 4. Запустите бота
python bot.py
